## [0.1.0] - 2026-03-29

### Added

- Primera versión funcional testeada del framework AI DS Project.
- Estructura base del repositorio con carpetas `demo/` y `templates/`.
- Plantillas de control y workbench para instanciación de casos.
- Script `create_case_instance.sh` para generar instancias de casos de demo.
- Casos de demo incluidos: `home-credit` y `pkdd99`.
- Documento `DEMO_WORKFLOW_STANDARD.md` con la dinámica estándar de trabajo (4 prompts operativos, bucle control/workbench, gestión de históricos).
- Ficheros de licencia: `COPYING` (GPL-3.0) y `COPYING.LESSER` (LGPL).
- Notas de conectividad (`CONNECTIVITY_LIBRARY_NOTES.md`) y skills publicadas (`PUBLISHED_SKILLS_NOTES.md`).
