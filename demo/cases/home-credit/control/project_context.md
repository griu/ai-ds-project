# Home Credit — Contexto del caso

Caso de demo orientado a riesgo de impago con dataset tipo Home Credit Default Risk.

Objetivo de la demo:
- mostrar un flujo DS gobernado;
- separar claramente control y workbench;
- usar un caso con suficiente riqueza tabular y dificultad analítica;
- demostrar framing, definición de muestras, EDA, monotonicidades, modelado, explicabilidad y preparación para productivización.
