# Project context - Home Credit Default Risk

## Objetivo de negocio
Priorizar solicitudes con mayor riesgo de impago y mejorar la consistencia del análisis previo a concesión.

## Tipo de problema
Clasificación supervisada de riesgo de default.

## Cómo se quiere usar el modelo
Como score de apoyo a decisión y priorización de revisión manual.

## Restricciones conocidas
No usar variables con leakage temporal ni asumir disponibilidad universal de todas las tablas históricas.
