# Supuestos operativos del modelo — Home Credit

## Unidad de decisión esperada
Solicitud actual.

## Pregunta a resolver
¿Se puede estimar el riesgo relativo de no repago de una nueva solicitud usando información conocida antes o en el momento de la solicitud?

## Uso esperado del output
- priorización de revisión;
- segmentación de riesgo;
- soporte a pricing o política de admisión en una fase posterior.

## Requisitos para las siguientes fases
- dejar claramente documentado qué datos son observables en fecha de solicitud;
- distinguir features snapshot vs features agregadas históricas;
- preparar una estrategia de validación que evite optimismo artificial;
- dejar la puerta abierta a explainability y monitorización.
