# Adquisición de datos para la demo — Home Credit

## Opción simple para demo
Trabajar con una muestra local descargada previamente desde Kaggle y copiada a:
- `inputs/data_samples/`

## Opción más didáctica
Trabajar solo con los documentos y una muestra reducida de tablas, retrasando la conectividad completa a la segunda fase.

## Nota
En esta demo se asume que el acceso real y el `pyproject.toml` se prepararán como tarea posterior, no antes del framing.
