# Project context - PKDD99 Financial

## Objetivo de negocio
Definir un proyecto temporal de riesgo / comportamiento financiero con observaciones repetidas por cliente y cuenta.

## Tipo de problema
Modelización supervisada con fuerte componente temporal y agregación relacional.

## Cómo se quiere usar el modelo
Soporte a priorización y análisis de comportamiento a partir de historiales.

## Restricciones conocidas
Respetar la direccionalidad temporal y evitar leakage derivado de tablas posteriores al evento objetivo.
